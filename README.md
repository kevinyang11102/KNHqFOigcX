# 基于Web的旅游管理系统 java908

## 项目概述

本项目是基于前后端分离架构的旅游管理系统，使用Java语言开发，结合SpringBoot、MyBatis等后端框架以及Vue3等前端框架，旨在提供高效、易用的旅游服务解决方案。

## 技术栈

- **开发语言**：Java
- **后端框架**：SpringBoot、MyBatis
- **前端框架**：Vue3
- **编辑器**：IDEA
- **数据库**：MySQL
- **数据库管理工具**：Navicat
- **JDK版本**：JDK8
- **构建工具**：Maven3+

## 功能模块

### 系统角色

- 管理员
- 用户

### 管理员功能

- 登录
- 个人中心
  - 个人信息
  - 修改密码
- 用户管理
- 景点类型管理
- 景点信息管理
- 景点评价
- 门票购买管理
- 旅游咨询管理
- 退出

### 用户功能

- 注册登录
- 个人中心
  - 个人信息
  - 修改密码
- 首页
  - 轮播图
  - 景点信息
  - 景点资讯
- 景点信息分类展示
- 景点详情
- 景点收藏
- 景点评论
- 门票购买
- 景点资讯
- 我的收藏
- 我的门票
- 退出
